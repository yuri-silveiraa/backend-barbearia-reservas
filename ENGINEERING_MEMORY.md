# Engineering Memory

Memória operacional deste repositório `backend-barbearia-reservas`.

Objetivo:

- registrar decisões de Clean Architecture, contratos e infraestrutura
- documentar erros de autenticação, CSRF e banco
- preservar soluções de testes e validações
- permitir continuidade entre máquinas trabalhando no backend
